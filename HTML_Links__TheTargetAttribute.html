Target Attribute in the HTML Links

The Target attribute specifies where to open the linked document. The Target Attribute having the following values

            
  • 
                _blank ---> Opens the Linked document in the new window or tab.
            
    • 
            
  • 
                _self ---> Opens the Linked document in the same window/tab as it was clicked (this is default)
            
    • 
            
  • 
                _parent ---> Opens the Linked document in the parent Frame.
            
    • 
            
  • 
                _top ---> Opens the Linked document in the full body of the window.
            
    • 
            
  • 
                Opens the Linked document in the named frame.
