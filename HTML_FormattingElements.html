Formatting HTML Elements

  1. Bold Text will be represented with the help of "b" tag

  2. Strong Text will be represented using "strong" tag

  3. Italic Text will be represented using the "i" tag

  4. Emphasized Text will be represented using the "em" tag

  5.             The "strong" and "em" notifies the text as "IMPORTANT"

  6. Small Text will be represented using the "small" tag

  7. Mark Text will be represented using the "mark" tag. It is used to define a highlighted text or marked text.

  8. Del Text will be represented using the "del" tag. It is used to define the deleted text using

  9. "Ins" tag will be used to define the Inserted Text in the current paragraph and also it will represented using the "ins" tag

  10. "Sub" tag will be used to define the Subscripted Textfor our text

  11. "Sup" tag will be used to define the Superscripted Text
