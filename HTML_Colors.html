HTML Colors

HTML Colors are represented by the RGB(Red, Green, Blue). Each color value is defined between 0 to 255.

The HTML Colors are represented with the hex codes. These code limit is 6 digits. It has "#" as prefix. First two digits define the "RR -->Red" , Second two digits are define the "GG - - > Green" and last two digits define the "BB --->Blue"

Blue --> RGB(0,0,255)

Red --> RGB(255,0,0)
